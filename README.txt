Welcome To FlashCards!

About ------------------------
In this Application you can create Decks of FlashCards
once the deck u created you can create Cards that belong to that 
deck,each card will have a question and answer, once the card is created you
can start a quiz from the deck you have created.
this quiz will keep track of the correct answers you had , displaying it once the quiz is over.
--------------------
Installing -> 
.yarn Install
.yarn start
NPM should also work.

ENJOY!
